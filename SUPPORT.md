# Contributor Covenant Code of Conduct

## Support

### How to Get Help

If you need help with this project, please use one of the following methods:

#### Documentation
- Read the [README.md](README.md) for basic usage
- Check [EXAMPLES.md](EXAMPLES.md) for code examples
- Review [CONTRIBUTING.md](CONTRIBUTING.md) for development guidelines

#### Community Support
- Search existing [GitHub Issues](https://github.com/Rezakarimzadeh98/secure-password-generator/issues)
- Open a new [Discussion](https://github.com/Rezakarimzadeh98/secure-password-generator/discussions)
- Ask questions with the `question` label

#### Reporting Issues
- **Bugs**: Use the [Bug Report template](https://github.com/Rezakarimzadeh98/secure-password-generator/issues/new?template=bug_report.md)
- **Features**: Use the [Feature Request template](https://github.com/Rezakarimzadeh98/secure-password-generator/issues/new?template=feature_request.md)
- **Security**: Follow the [Security Policy](SECURITY.md)

### Response Time
- Issues: Within 48 hours
- Pull Requests: Within 72 hours
- Security Issues: Within 24 hours

### Before Asking for Help

Please ensure you have:
1. Read the relevant documentation
2. Searched existing issues and discussions
3. Tried the latest version
4. Provided clear reproduction steps

### What Information to Include

When asking for help, include:
- Your environment (OS, browser, Node.js version)
- Steps to reproduce the issue
- Expected vs actual behavior
- Error messages or screenshots
- Relevant code snippets
